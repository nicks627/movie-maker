# Political Playbook

## 向いている題材

- 国内政治
- 予算 / 税制 / 官庁解説
- 国際政治
- 政策対立や争点整理

## 企画の切り方

- 最初に `何が争点か` を一撃で置く
- 次に `誰が関与しているか`
- その後 `制度 / 数字 / 時系列`
- 最後に `なぜ今見るべきか`

## Script の作り方

- 解説動画よりも `対立軸` と `因果` を明示する
- 1 scene 1 論点を厳守する
- 数字・予算・制度は scene を分けて処理する
- 強い見出しと地図 / 比較 popup を組み合わせる

## レイアウトの考え方

- headline は強く、本文は冷静に
- `PoliticalShockBanner`, `TwoColumnComparison`, `GeoPoliticalMap`, `StatsCard` を優先
- subtitle / popup / 背景の競合を特に避ける

## 字幕 / popup / 背景の使い分け

- 字幕: 説明の主文
- popup: 争点、数字、比較、引用
- 背景: 会見、国会、地図、制度の文脈

## 音声 / BGM / SE

- 音で煽りすぎない
- `hook` と争点切替だけ音の変化を強める
- 数字説明は静かめ
- impact SE は場面の印として限定的に使う

## 素材収集

- 公式 source, 報道写真, 施設写真, 地図素材を優先
- stock は補助に回す

## Render 前 review

- 用語や数字の読みが不自然でないか
- popup の情報量が字幕を潰していないか
- CTA が扇動だけで終わっていないか

## よくある失敗

- headline が強すぎて本文が追えない
- 1 scene で数字と主張を詰め込みすぎる
- map / comparison を使わず、口頭だけで押し切る
