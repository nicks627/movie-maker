# Thumbnail Prompt

## Long

### Recommended overlay text

- 人事で支配
- 官邸主導の裏側

### Alternate overlay text

- 命令より怖い
- 空気が従う
- 情報まで寄る
- 人事で空気が変わる

### Prompt

Japanese political documentary YouTube thumbnail, hidden power structure inside Japan's Prime Minister's Office, night view of Kantei and National Diet, bold two-layer Japanese headline with the main phrase "人事で支配" and smaller subline about the hidden side of cabinet-led politics, large smartphone-readable typography, left-heavy editorial text block, right-side looming government architecture, red warning accents, white main text, dark navy and black base, subtle personnel chart arrows, official document fragments, faint bureaucrat silhouettes, tense investigative atmosphere, premium current-affairs visual language, realistic photo-illustration look, crisp contrast, simple composition, no clutter.

### Negative prompt

Crowded collage, too many symbols, tiny unreadable text, pastel palette, purple-heavy palette, comedy mood, exaggerated anime face, cheap tabloid look, blurry buildings, weak contrast, excessive document text, noisy neon effects, overdecorated background.

### Color direction

Deep navy and charcoal base, white main headline, controlled red warning accents, muted gray support elements, sharp high-contrast palette for mobile readability, avoid colorful variety-show tones.
